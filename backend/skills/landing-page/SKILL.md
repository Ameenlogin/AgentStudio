# Landing Page

Produce a high-converting, beautiful single-page site.

## Structure (in order)
1. **Nav** — logo left, 3–5 links, one primary CTA right.
2. **Hero** — one sharp headline (benefit, not feature), one sub-line, primary + secondary CTA, a product visual/mockup. Above the fold.
3. **Social proof** — logos or a short testimonial strip.
4. **Features** — 3 columns, icon + title + one sentence each. Show value, not specs.
5. **How it works** — 3 steps.
6. **Deeper feature / showcase** — alternating image+text rows.
7. **Pricing** — 2–3 tiers, highlight the recommended one.
8. **FAQ** — accordion, 4–6 items.
9. **Final CTA** — restate the promise + button.
10. **Footer** — links, legal, socials.

## Craft
- Headlines: specific and confident ("Ship in minutes, not weeks"), not vague.
- One accent color used for every CTA. Buttons look clickable and consistent.
- Whitespace-forward; sections breathe (80–120px vertical padding on desktop).
- Real-looking copy and placeholder imagery (SVG mockups if no assets).
- Fully responsive; CTAs reachable on mobile.
- Fast: inline critical CSS or one stylesheet, optimized SVGs.

## Deliverable
Build `index.html` (+ `styles.css`) in the workspace, run/preview it, then zip it for download if asked.
